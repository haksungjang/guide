# 오픈소스 가이드 \(draft\)

이 문서는 SK텔레콤에서 오픈소스를 사용, 기여 그리고 공개하는데 올바른 가이드를 제공하기 위해 작성하였습니다. 

## 오픈소스란 무엇인가요?

누군가 소프트웨어를 오픈소스로 공개했다는건 누구나 자유롭게 소프트웨어를 사용, 수정, 배포할 수 있음을 의미합니다. 이는 소프트웨어를 공개하면서 [오픈소스 라이선스](https://opensource.org/licenses)로 이러한 권한을 부여했기 때문입니다. 오픈소스 라이선스에 대해서는 오픈소스 사용하기에서 자세히 알아보겠습니다.
