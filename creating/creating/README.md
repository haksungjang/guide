# 오픈소스 공개하기

SK텔레콤은 오픈소스가 소프트웨어 개발 문화를 혁신하는 훌륭한 도구임을 분명히 알고 있으며, SK텔레콤의 오픈소스 커뮤니티 참여를 적극적으로 지원합니다. 공개하는 오픈소스 프로젝트에 법적, 윤리적, 기술적인 문제가 발생하지 않고, 오픈소스 협업의 이점을 최대한 활용할 수 있도록 가이드를 충실히 준수하기 바랍니다.
